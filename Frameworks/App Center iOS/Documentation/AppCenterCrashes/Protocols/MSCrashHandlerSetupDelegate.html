MSCrashHandlerSetupDelegate

@protocol MSCrashHandlerSetupDelegate <NSObject>

This is required for Wrapper SDKs that need to provide custom behavior surrounding the setup of crash handlers.

  • -shouldEnableUncaughtExceptionHandler

    Callback method that gets a value indicating whether the SDK should enable an uncaught exception handler.

    @discussion Do not register an UncaughtExceptionHandler for Xamarin as we rely on the Xamarin runtime to report NSExceptions. Registering our own UncaughtExceptionHandler will cause the Xamarin debugger to not work properly (it will not stop for NSExceptions).
